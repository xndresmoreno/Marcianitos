# Marcianitos
Juego creado a partir de space invaders.

Caracteristicas principales
  Movimiento fluido del jugador (derecha/izquierda)
  Disparo normal y disparo triple obteniendo un buff
  Oleadas progresivas con dificultad progresiva
  Jefe final con barra de vida cada 5 niveles
  Un sistema de puntuacion y guardado automatico
  Animacion de explosiones y un menu con demostracion de fondo

Requisitos 
Antes de ejecutar el juego, hay que asegurarse de tener instalado:
  Python 3.8 o superior
  Libreria Pygame ("pip install pygame" en bash para installar)
